### PLC使用说明
PLCSDK主要分为三个方面：通过opc访问，通过tcp访问，通过udp访问
## 一、OPC连接(Demo 请用OPCDemo为启动首选项)
opc连接主要功能集中在OPCClient类中：（OPC的UA方式，需引用两个Opc.Ua的dll）
1、开放字段：LogTxt 此字段是一个黑匣子缓存，每次读取，写入，出现错误的时候写入。
2、使用方法：此类实现了单例模式，通过Instance可直接访问。
	  1、初始化：OPCClient.Instance.InIt(string ip,int port,false); 
	       如果采取opc的方式连接plc，在程序开始时候需要首先实现InIt()方法，实现初始化。
		   三个参数分别是opc的ip，opc的端口，是否启动黑匣子。
	  2、调用：Read(),write()
	       在此类中主要实现了写入和读取，其中两个异步，两个单线程同步。
3、说明：
      注意在此类初始化时会自动检测设备是否正常（2s每轮）。若设备掉线，会将设备数据收集模式设置为禁用。
	  此类没有订阅方法，是考虑到OPC会丢数据的不稳定性。在生产过程中实时要求高的数据不建议使用OPC
	  的方式连接，同时也欢迎共同解决OPC丢数据的问题。
